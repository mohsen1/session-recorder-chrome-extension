A Chrome extension that captures a full, multi-modal recording of a web app — interactions, network, screenshots, voice, annotations — and exports a fully local report.md optimized for a coding agent to digest.

Download the extension View on GitHub

Manifest V3 · WXT + React + TypeScript · Nothing leaves your machine.

How it works

  1. 01

    Hit record

    Open the side panel and start recording. The extension attaches to the tab and captures every stream through one funnel.

  2. 02

    Exercise the app

    Reproduce the bug. Narrate out loud, drop markers, annotate the screen. Everything persists to IndexedDB as you go.

  3. 03

    Export the report

    Pick a verbosity level with a live token estimate and export a local zip. Hand report.md to your coding agent.

What it captures

Full network capture

Request and response bodies, headers, timing, and websockets via chrome.debugger / CDP — plus console logs, exceptions, and failed requests.

Screenshots

Page.captureScreenshot at a configurable frequency, with near-duplicate frames automatically deduped so the timeline stays lean.

Real-time voice narration

Narrate while you record via an offscreen MediaRecorder, segmented for optional cloud transcription — OpenAI-compatible, Deepgram, or ElevenLabs Scribe.

Annotations

An in-page shadow-DOM canvas overlay for vector shapes, saved alongside the annotated screenshot. Never trimmed — it is an explicit signal of what mattered.

Intelligent trimming

Captured at full fidelity, trimmed deterministically at export into four levels — Full, Standard, Compact, Minimal — each with a live token estimate. No LLM in the loop.

Redaction on by default

Auth headers, token-like fields, sensitive URL params, and password inputs are masked at capture time — raw secrets never touch disk.

Install (load unpacked)

Download the zip and unpack it, or build from source. Then load it into Chrome as an unpacked extension.

  1. Visit chrome://extensions and enable Developer mode.
  2. Click Load unpacked and select the chrome-mv3 folder.
  3. Click the toolbar icon to open the side panel and hit Record.

While recording, Chrome shows a "… is being debugged" banner — that is expected; it is how the extension taps the full network and console streams.
